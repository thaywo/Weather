# Weather application

## Project setup
You should have a basic knowledge of Javascript, NodeJS and npm before setting this up. 
Steps are given below:
```
- npm i -g @vue/cli
- vue create {{filename}}
```

### Starting the local server
This website require a local server. So, for that just do:
```
npm run serve
```
